- Installed Firebase SDK (`npm install firebase`)
- Created `src/lib/firebase.js` — config file with project credentials
- **Auth:**
  - Login screen with Google sign-in + email/password option
  - Email: sign in / create account toggle, friendly error messages
  - Auth state listener in App.jsx (loading → login → app)
  - Sign out button + user info (avatar, name, email) in Settings
- **Firestore sync:**
  - `Storage.syncToFirestore(userId)` — writes all syncable data to `users/{userId}` doc
  - `Storage.pullFromFirestore(userId)` — reads cloud data on login
  - `Storage.mergeFromCloud(cloudData)` — writes cloud data into localStorage
  - `setDataAndSync` wrapper — every `setData` call also triggers background Firestore sync
  - On first login with existing localStorage data: pushes local → cloud
  - On login to new device (empty localStorage): pulls cloud → local
  - "Syncing…" toast while pulling from Firestore
- **Security:** Firestore rules lock each user to their own `users/{userId}` path
